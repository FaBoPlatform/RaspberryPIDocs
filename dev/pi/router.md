# Wifi Routerの設定

Wifi Routerを追加し、WifiのスポットからRassberryPIを見えるようにする。

## 利用可能な機材

|商品名|概要|
|:--|:--|
|ちびファイ2 MZK-UE150N|USBでRasPIに接続する事でeth1として識別|
|ちびファイ2ac MZK-UE450AC|RaspPIとWifi接続|
|ちびファイ MZK-RP150NA|有線でRaspPIで接続|
